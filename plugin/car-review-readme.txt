Spice up your sidebar and footer with recent posts plugin widget, easily. 

== Description ==
Hello. I’m a plugin called Terra Motive – Car_Review. This plugin gives you the ability to showcase specific posts on WordPress, from a category into a sidebar or widget. 
	
== Installation ==
This section describes how to install the plugin and get it working.

1. Upload the everything in folder called “Car-Review” to the “/wp-content/plugins/” directory
2. Activate the plugin through the Dashboard > Plugins menu on WordPress
3. Place `<?php do_action('plugin_name_hook'); ?>` in your templates

== Usage ==
This section describes how to display recent posts by category name with the plugin 

1. To begin open up Dashboard > Car Review > Add New Post
2. Fill out the designated area to create your very own custom post 
3. Ensure your custom post has a “Feature Image” thumbnail to enhance overall look of widget  
4. On the right hand column under “Type”, pick “Custom” as the category slug to display the post 
5. The custom query only allows WP to retrieve ‘3’ post under that specific category as standard, which user can replace under “functions.php” 
7. Publish the your custom post and click on “View” to view your post in browser 
6. You're Done! Enjoy your new custom category post widget :)
